# ICS_Export

Beschreibung

Dieses Script erstellt eine .ics-Datei aus einem aus ATOSS Staff Efficiency Suite exportierten Mitarbeiterdienstplan. Die .ics-Datei kann anschließend in ein Kalender-Programm wie Google Kalender oder Apple Kalender importiert werden.

Funktionalität

Liest die zur Terminerstellung relevanten Daten aus dem Excel-Worksheet und erstellt eine .ics-Datei mit den Arbeitszeiten und Abwesenheiten.

Anforderungen

Excel 2013 und höher oder Libre Office <br> ATOSS-Export-Datei im Excel-Format

Öffne das Script im VBA-Editor, ändere die Variable icsFile auf den gewünschten Dateipfad und führe das Script aus.
